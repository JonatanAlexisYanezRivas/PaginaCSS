Programación Web

¿Qué es?

La programación web es la creación de sitios dinámicos y estaticos en Internet. La creación de estos sitios depende de lenguajes de programación enfocados especialmente a la programación web en cuanto a aparencia, esta parte se le conoce como el frontend

Además de la apariencia de un sitio web también es importante mencionar que para que trabaje necesita lenguajes enfocados a lo que es la lógica de la página y el como va a funcionar, esta parte se le conoce como el backend

Este lenguaje de programación Web fue inventado por Tim Berners-Lee en 1986. El HTML combina el concepto de Hipertexto, que permite conectar dos elementos entre si, y el SGML (Lenguaje Estándar de Marcación General) que permite colocar etiquetas o marcas en un texto para indicar como queremos que ese documento se visualice.

Lenguajes de programación web

HTML

Lenguaje de marcado HTML utilizado en el desarrollo web para crear el contenido y la estructura de una pagina web

CSS

Hojas de estilo en cascada, es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado

JavaScript

Es un lenguaje de programación interpretado. Se define como orientado a objetos, basado en propototipos, imperatuvo, débilmente tipado y dinámico. Usado para añadir características interactivas a tu sitio web
